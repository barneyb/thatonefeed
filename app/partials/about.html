About

That One Feed is a super simple app designed to address some downsides to the main Feedly web interface.

The front end is constructed using AngularJS, a lightweight JavaScript framework constructing for rich applications.

The server side is nothing more than a thin proxy to the Feedly Cloud, running Railo. There is no database; all storage happens directly in the Feedly Cloud.

The entire application's lifecycle is managed via Grunt with the help of Node, NPM, and a bunch of plugins.
